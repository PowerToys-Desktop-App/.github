## 🚀 Overview
PowerToys for Windows is a free collection of utilities from Microsoft that helps you get more out of your PC. It bundles a set of tools that tune, streamline, and extend everyday Windows tasks — arranging windows into custom layouts, launching apps and files in an instant, picking colors, resizing images, renaming files in bulk, and even copying text straight off your screen. A single Microsoft PowerToys download adds all of these utilities to Windows 11 and Windows 10, each one toggled on or off from a central settings dashboard. Whether you're a developer, a designer, or simply a power user, it gives you a faster, more customizable desktop.

## ✨ Features
- **FancyZones** — Snap windows into custom grid layouts so you can organize your screen exactly how you like.
- **PowerToys Run** — A fast launcher that opens apps, files, and searches with a quick keyboard shortcut.
- **Text Extractor** — Copy text from anywhere on screen — images, videos, or menus — with built-in optical character recognition.
- **Color Picker** — Grab the exact color of any pixel on your display and copy it in the format you need.
- **Image Resizer** — Resize one or many images right from the File Explorer right-click menu.
- **PowerRename** — Rename large batches of files at once using search, replace, and pattern matching.
- **Keyboard Manager** — Remap keys and create custom shortcuts to match the way you work.

## ❓ FAQ
**Is PowerToys free?**
Yes. PowerToys is a free, open-source project from Microsoft. You can download and use every utility at no cost, with no subscription required.

**What is PowerToys Text Extractor?**
PowerToys Text Extractor is a built-in tool that uses optical character recognition to copy text from anywhere on your screen — including images, videos, and windows that don't normally let you select text. Press its shortcut, drag over the area, and the text is copied to your clipboard.

**Which versions of Windows are supported?**
PowerToys runs on modern 64-bit versions of Windows, including Windows 10 (version 2004 or later) and Windows 11. Keeping Windows up to date helps all of the utilities run smoothly.

**Is my data secure?**
PowerToys runs locally on your own PC, and its utilities work directly on your files, windows, and clipboard without sending them elsewhere. You stay in control of what each tool can access.
